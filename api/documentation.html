Authentication

All API requests require authentication using an API key. Include your API key in the request header:

Request Header 
Authorization: Bearer YOUR_API_KEY

Getting an API Key: Sign up for an account and generate your API key from the account settings page.

GET /etfs/{ticker}/history

Get historical price data for an ETF

Parameters:
Parameter Type Description
ticker string ETF ticker symbol
period string Time period: 1d, 5d, 1m, 3m, 6m, 1y
interval string Data interval: 1m, 5m, 15m, 1h, 1d
GET /distributions

Get upcoming distribution dates for all ETFs

Query Parameters:
Parameter Type Description
days integer Number of days ahead (default: 30)
ticker string Filter by specific ticker

Code Examples

JavaScript

// Fetch ETF data
async function getETFData(ticker) {
    const response = await fetch(`https://api.mstymillionaire.com/v1/etfs/${ticker}`, {
        headers: {
            'Authorization': 'Bearer YOUR_API_KEY'
        }
    });
    
    const data = await response.json();
    console.log(`${ticker} Price: $${data.price}`);
    console.log(`${ticker} Yield: ${data.yield}%`);
}

getETFData('MSTY');

Python

import requests

# Set up headers
headers = {
    'Authorization': 'Bearer YOUR_API_KEY'
}

# Get ETF data
response = requests.get(
    'https://api.mstymillionaire.com/v1/etfs/MSTY',
    headers=headers
)

data = response.json()
print(f"MSTY Price: ${data['price']}")
print(f"MSTY Yield: {data['yield']}%")

Error Handling

The API uses standard HTTP status codes and returns JSON error responses:

400 Bad Request - Invalid parameters
401 Unauthorized - Invalid or missing API key
404 Not Found - Resource doesn't exist
429 Too Many Requests - Rate limit exceeded
500 Internal Server Error
Error Response Format:
{
  "error": {
    "code": "INVALID_TICKER",
    "message": "The ticker symbol 'XYZ' is not valid",
    "status": 400
  }
}
